# Security Policy

## Supported Versions

Only the latest release and the current beta are actively supported with security fixes.

| Version | Supported |
| ------- | --------- |
| Latest stable | :white_check_mark: |
| Latest beta | :white_check_mark: |
| Older versions | :x: |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report vulnerabilities privately via [GitHub Security Advisories](https://github.com/mstrhakr/compose_plugin/security/advisories/new).

Include as much detail as possible:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

You can expect an acknowledgement within 48 hours and a resolution or status update within 7 days.
